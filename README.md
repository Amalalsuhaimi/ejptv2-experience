#	نبذة عن ejptv2 

اختبار اختراق (عملي) مصمم للمبتدئين في عالم الاختراق. مقدم من INE بالتعاون مع    elearnsecurity
تحتوي النسخة الثانية من الاختبار علي ٣٥ سؤالاً و مدة الاختبار يومين ( ٤٨ ساعة ) . نوع الاختبار open-book
أي تستطيع العودة للمنهج او ملاحظاتك و استخدام قوقل أيضاً (لن تجد إجابة مباشرة لأسذلة الاختبار و لكن قد تجد بعض الأدوات و الاجراءات المساعدة في ذلك). لا يوجد متطلبات معينه لأخذ الشهادة، ولكن يفضل ان يكون هناك معرفة سابقة بأساسيات الشبكات، أنظمة التشغيل وأحد لغات البرمجة.
